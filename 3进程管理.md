## 进程管理的命令
当一个应用程序启动时，就可以把它理解成为一个进程
### `who` 命令
查看当前是谁在使用    
+ `tty`
tty7代表的是Linux的桌面终端  
tty1-tty6都是文字终端  
+ `tps`
tps/0 代表的是设备终端，即为Linux下的窗口
+ `PID`
`PID` 当前程序的进程ID
### `ps`命令
+ 参数
  + `ps a` 列出当前的所有用户的信息
  + `ps u` 显示用户自身的信息
  + `ps x`
